

COMMAND + O         -> NAVEGA PARA UMA CLASSE - ADICIONANDO :{NUMERO VAI DIRETO NA LINHA ESPECIFICADA}
COMMAND + SHIFT + O -> NAVEGA PARA UM ARQUIVO  - ADICIONANDO :{NUMERO VAI DIRETO NA LINHA ESPECIFICADA}
COMMAND + ALT + O   -> NAVEGA PRA UM SIMBOLO

COMMAND + SHIFT + F -> FIND IN PATH

SHIFT + SHIFT       -> FIND ANYTHING IN EVERYWHERE -> TAB MUDA A TAB DE PROCURA -> ALT + ARROW UP e ALT + ARROW DOWN MOVES THROUGH HISTORY

COMMAND + E         -> RECENT FILES -> COMMAND + E + ENTER SWTCH BETWEEN GO TO LAST
COMMAND + SHIFT + E -> RECENT EDITED FILES

COMMAND + [ -> NAVIGATE BACK
COMMAND + ] -> NAVIGATE FORWARD

COMMAND + 1          -> GO TO PROJECTS -> START TYPING TO SEARCH FILES IN IT
COMMAND + ARROW DOWN -> JUMP FROM PROJECT STRUCTURE TO SOURCE CODE

COMMAND + N          -> CREATE A NEW FILE


COMMAND + SHIFT + ARROW DOWN -> MOVE LINE DOWN
COMMAND + SHIFT + ARROW UP   -> MOVE LINE UP

COMMAND + SHIFT + F12        -> HIDE/SHOW ALL THE WINDOWS BUT THE EDITOR

COMMAND + SHIFT + ARROWS     -> RESIZE WINDOW (MUST NOT BE FOCUSED IN EDITOR)


ALT + ARROW UP               -> SELECT A WORD, THEN THE LINE, THEN THE
ALT + ARROW DOWN             -> SHRINK SELECTION

COMMAND + SHIFT + V -> SHOWS LASTS COPPIED/CUTTED CODE

COMMAND + BACKSPACE -> DELETE LINE

CONTROL + G -> FIND NEXT OCCURRENCE AND GIVES IT A NEW CURSOR
CONTROL + SHIFT + G -> FIND ALL OCCURRENCES AND GIVES EACH A NEW CURSOR

SHIT + (function) F6 -> RENAME FILE, CLASS OR METHOD

COMMAND + SHIFT + G -> FIND NEXT OCCURRENCE

CONTROL + ALT + I -> INDENT SELECTED PIECE OF CODE
COMMAND + ALT =L  -> INDENT THE WHOLE FILE

ANT + ENTER -> SHOW LANGUAGE INFECTION AND OTHER CODE INDENT OPTIONS

COMMAND + P -> SHOW METHODS PARAMETERS

ALT + ENETER -> SMART COMPLETE (LIKE CTRL + 1 IN ECLIPSE)

CONTROL + ALT + SPACE -> SMART AUTO COMPLETE (Ex.: valueOf + this command = shows all valueOf in external classes)


ALT + / -> HIPPIE COMPLETE -> COMPLETE WITH ANYTHING THAT STARTS WITH INSIDE YOUR CLASS

COMMAND + J -> SHOW TEMPLATES

COMMAND + ALT + T -> SURROUND WITH (TRY CATCH, IF< ET)

FUNCTION + COMMAND + ARROWS -> NAVIGATE TO END OR BEGINNING OF THE FLE

CONTROL + R -> RUN LAST EXECUTION
CONTROL + ALT + R -> SHOW LAST EXECUTIONS TO RUN
